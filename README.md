# Sales Dashboard

A comprehensive data analytics project designed to track, analyze, and visualize sales performance. This repository features an interactive **Power BI Dashboard** built using a sample sales dataset to extract actionable business insights, identify key trends, and monitor performance metrics.

## 🛠️ Tech Stack & Tools

* **Business Intelligence:** Power BI Desktop (`.pbix`)
* **Data Source:** CSV Spreadsheet (`sales_data_sample.csv`)
* **Data Transformation:** Power Query (ETL processing)
* **Modeling & Metrics:** DAX (Data Analysis Expressions)

---

## 📈 Key Features & Insights

* **KPI Performance Cards:** Real-time visibility into Total Revenue, Total Quantity Sold, Average Order Value, and Total Orders.
* **Sales Trends Over Time:** Time-series analysis showcasing monthly and quarterly sales growth patterns to pinpoint peak seasons.
* **Product Line Analytics:** Deep dive into top-performing products, identifying which categories drive the highest profit margins.
* **Geographical Mapping:** Regional breakdown highlighting top revenue-generating countries and cities.
* **Customer Segmentation:** Analysis of order distribution, status tracking (Shipped, In Process, Cancelled), and deal size segments (Small, Medium, Large).
